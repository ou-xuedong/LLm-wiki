## 改造模式（提醒）

本站走**模式 1：渲染层** —— MD 是源，HTML 只是浏览器视图。

主人消费层的真正 HTML 报告（如卟卟鸡 `评估记录/*.html`）走**模式 3：替换**——直接生成 HTML，不在本站索引内。

详见 [HTML 优先输出范式 · 主人项目改造指引](知识库/wiki/insights/HTML优先输出范式_主人项目改造指引.md)
