# 金融×AI 机会点孵化器

每天和 AI 讨论一个「金融领域结合 AI 的机会点」,记录反馈,系统自进化、越来越懂你。纯 markdown,单人用,本文件夹独立运行。

## 怎么用

开会话说一句 **"今天聊一个"** 或 **"看草稿"**,Agent 会:
1. 读你的画像 → 抛一个金融×AI 机会点(去过重)
2. 和你深聊:机会点 / 为什么现在 / 切入角度 / 风险
3. 把讨论和你的反馈记进 `日志/`
4. 更新你的偏好画像,下次更懂你
5. 如果你的判断和过去矛盾,会反问你

Agent 的完整操作规则见 `CLAUDE.md`。

## 文件

- `画像.md` — 你的偏好 / 已覆盖方向 / 矛盾台账(系统对你的理解)
- `索引.md` — 所有聊过的话题速查(去重 + 查矛盾)
- `日志/` — 每天一个话题文件,完整讨论记录
- `周报.md` — 每周摘要(老话题压缩在这,省 token)

## 机制(都是土办法,零依赖)

- **去重**:读索引清单判重,不用向量库
- **自进化**:偏好写成自然语言,每次注入
- **矛盾**:态度翻转就反问你,不闷头覆盖
- **压缩**:三层记忆,只读最近 7 天,token 恒定
