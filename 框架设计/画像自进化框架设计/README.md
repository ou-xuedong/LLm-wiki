# 画像自进化框架

**版本：v1.0.0**（2026-06-15 首版）　|　抽象来源：planX 实例

> 一套用于**长期、跨多次会话深度研究某个对象**（人/客户/对手/关系/议题）的目录与方法框架。
> 让 AI 对该对象的认知**越积越准、可溯源、且不爆 token**。

---

## 一句话原理

**信息四层模型 + 画像累积刷新 + 分层加载** ——
同一信息只在一层权威存放（防膨胀/回音壁），画像随每次讨论就地刷新（活体记忆），每次会话只读"规则+画像+最近1次总结"（固定开销、与讨论次数无关）。

## 四层模型速览

| 层 | 装什么 | 增长 |
|---|---|---|
| ① 贴源层 | 原始素材，只记录不分析 | 按次追加 |
| ② 讨论总结 | 一次一文件，复盘洞察 | 一次一文件 |
| ③ 人物画像 | 当前最新结论（长期记忆） | **累积刷新** |
| ④ 产出文件 | 拿去就用的物料 | 可清理 |

## 两个用途

- **快速创建**：拷模板 → 替占位符 → 第一次讨论走三件套。见 `使用指南.md §一`。
- **升级改造**：读 CHANGELOG → 认旧版本 → 备份 → 分类改动 → 升版本号 → 验证。见 `使用指南.md §二`。

## 适用边界

✅ 长期多轮、需累积认知的对象研究。
❌ 一次性任务、无连续性的零散查询。
