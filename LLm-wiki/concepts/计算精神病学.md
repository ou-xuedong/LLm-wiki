# 计算精神病学


## 关联

**上位概念**：认知神经科学, 强化学习
**相关概念**：认知灵活性, 强化学习_预测误差, 目标导向系统, 习惯系统, 规则切换

# 🧠 计算精神病学（Computational Psychiatry）

## 1. 定义

计算精神病学是认知神经科学与精神疾病交叉的前沿领域，其核心思想是：将精神疾病症状重新概念化为**认知算法参数的异常偏差**，用强化学习（RL）等计算模型对人类认知过程进行数学建模，从而实现对精神疾病的**量化诊断、机制解释和精准干预预测**。

**学科定位**：传统精神病学基于症状描述（DSM/ICD），计算精神病学基于机制模型（Computational Mechanism）。两者互补而非替代。

---

## 2. 核心方法论

### 2.1 三层闭环研究范式

```
行为实验（概率反转学习、延迟折扣等任务）
    ↓
计算模型拟合（估计个体化参数：α、λ、γ等）
    ↓
神经验证（fMRI/EEG 脑区激活与参数的关联分析）
    ↓
因果调控（HD-tDCS/TMS 验证神经-行为的因果关系）
```

### 2.2 关键认知任务范式

| 任务名称 | 测量的认知功能 | 代表应用 |
|---|---|---|
| **概率反转学习任务**（Probabilistic Reversal Learning）| 认知灵活性、规则切换、PE 信号 | OCD、抑郁 |
| **延迟折扣任务**（Delay Discounting）| 冲动控制、未来奖励折现 | ADHD、成瘾 |
| **反转学习任务**（Extinction Learning）| 恐惧消退、安全学习 | PTSD、焦虑 |
| **手指敲击任务**（Bandit Task）| 探索-利用平衡、强化学习 | 帕金森、精神分裂 |

### 2.3 核心计算模型：强化学习（RL）

计算精神病学最常用的模型框架是**强化学习的预测误差模型**：

**经典 Rescorla-Wagner 模型**：
```
δ_t = r_t - V(s_t)           （预测误差）
V(s_t) ← V(s_t) + α · δ_t   （价值更新）
```

**带习惯权重的扩展模型**（由 Tom Verguts 等人提出，陈琦团队在 OCD 研究中广泛使用）：
```
habit_strength = λ · Pavlovian_signal
final_output = (1-λ) · goal_directed + λ · habitual
```

**关键参数及认知含义**：

| 参数 | 符号 | 正常范围 | 异常情况 | 代表疾病 |
|---|---|---|---|---|
| 学习率 | α | 0.1-0.5 | 偏低（更新太慢）或偏高（过度反应）| OCD、抑郁 |
| 折扣因子 | γ | 0.9-0.99 | 偏低（只看眼前）| ADHD、成瘾 |
| 习惯权重 | λ | 0.1-0.3 | 偏高（习惯主导）| OCD、帕金森 |
| 温度参数 | τ | 0.5-2.0 | 偏高（过度探索）/偏低（过度利用）| 精神分裂、抑郁 |

## 3. 主要研究方向与应用疾病

### 3.1 强迫症（OCD）

**核心发现**（陈琦团队系统性贡献）：
- OCD 患者的 **λ（习惯权重）显著偏高**，Pavlovian 习惯系统对目标导向工具性行为的调节**减弱**
- 纹状体尾状核与壳核的**功能分离异常**——两区域对奖励/惩罚的区分变模糊
- 目标导向皮层-纹状体学习系统在 OCD 患者及其一级亲属中均存在损伤，提示是**内表型**（endophenotype）

**机制解释**：
正常情况下，前额叶根据结果反馈动态调整行为策略。但 OCD 患者中，习惯系统权重过高，压制了目标导向系统的校正信号，导致重复行为无法被"结果不好"的反馈所抑制——解释了为什么患者"知道没必要，但停不下来"。

### 3.2 抑郁症

**核心发现**：
- 抑郁患者的**负向预测误差（negative PE）神经表征减弱**——负反馈进入前额叶-MFC 系统的信号变弱
- 前额叶对负向预测误差的响应降低，导致学习率（α）异常，无法有效从失败中更新策略
- 这与抑郁症"习得性无助"的临床表现高度吻合——不是不想改变，而是无法从负面反馈中学习

**计算模型解释**：
```python
# 正常人的负向PE响应
δ_negative = r_actual - V(s)  # < 0 → V降低 → 行为抑制

# 抑郁患者
δ_negative = 0.3 × (r_actual - V(s))  # 衰减系数 → 学习不足 → 行为不变
```

### 3.3 成瘾与ADHD

两者共享一个核心计算异常：**折扣因子 γ 偏低**。

| 疾病 | 具体表现 | 计算异常 |
|---|---|---|
| **成瘾** | 偏好即时奖励，无法延迟满足 | γ ≈ 0.5-0.7（正常 > 0.9）|
| **ADHD** | 冲动决策，无法等待 | γ 极低，τ（温度）偏高 |

成瘾的神经基础是**多巴胺系统的 PE 信号异常**——对即时奖励的 PE 放大，对延迟奖励的 PE 衰减。

## 4. 与传统精神病理学的关键差异

| 维度 | 传统精神病学 | 计算精神病学 |
|---|---|---|
| **诊断逻辑** | 症状归类（类别诊断）| 参数偏差（维度诊断）|
| **解释力** | 相关性描述 | 因果机制模型 |
| **个体化** | 粗分类（正常/异常）| 连续参数空间 |
| **预测力** | 弱，难以预测治疗反应 | 强，可模拟干预效果 |
| **干预设计** | 试错 | 模型驱动的精准干预 |

---

## 5. 对 AI Agent 工程的迁移价值

计算精神病学的核心贡献是揭示了**人类在不确定环境中的决策算法**。这对于构建 AI Agent 的决策系统有直接迁移价值：

### 迁移一：探索-利用平衡
人类认知灵活性的计算模型，直接对应 AI Agent 在"利用已知最优策略"和"探索新策略"之间的切换难题。陈琦的参数化框架（λ、α、γ）提供了一个**可量化的切换阈值系统**。

### 迁移二：异常状态检测
计算精神病学告诉我们：当 RL 参数偏离正常范围时，人会表现出"认知障碍"。同理，当 Agent 的 Q-value 更新率（α等效）、探索率（τ等效）出现异常模式时，可以触发"系统状态异常"的主动干预。

### 迁移三：可解释性
传统 RL 是黑箱。计算精神病学提供了"认知归因"的思路——不仅告诉用户"该怎么做"，还能解释"为什么你（人类）倾向于这样做"。
