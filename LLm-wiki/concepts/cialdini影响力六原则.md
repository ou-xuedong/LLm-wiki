# 🧠 Cialdini影响力六原则

## 1. 定义与核心原理

社会心理学家 Robert Cialdini 于 1984 年在《影响力》（*Influence*）中提出的六大说服原则，后在 2016 年《Pre-Suasion》中补充第七原则。这些原则是人类在信息过载下依赖的心理捷径（Heuristics）。

**七大原则**：
1. **互惠**（Reciprocity）：先给后取
2. **承诺与一致**（Commitment & Consistency）：公开承诺强化后续行为
3. **社会证明**（Social Proof）：从众、跟风
4. **喜好**（Liking）：相似/欣赏的人推荐更可信
5. **权威**（Authority）：专家、机构、徽章
6. **稀缺**（Scarcity）：越少越珍贵
7. **团结**（Unity，2016 新增）：同类身份认同高于喜好

## 2. 关键技术路线与变体

与 Kahneman、Thaler 的行为经济学共振，常联合使用：
- **锚定效应**（Anchoring）：先给高价再打折
- **损失厌恶**（Loss Aversion）：失去比获得感受强 2 倍
- **心理账户**（Mental Accounting）：免邮 ≠ 等额降价
- **峰终定律**（Peak-End Rule）：体验记忆由峰值+终点决定
- **默认效应**（Default Effect）：Thaler 助推理论核心

## 3. 行业应用场景

**在亚马逊投流中的武器化映射**：

| 原则 | 投流抓手 |
|---|---|
| 社会证明 | 评论数、星级、BSR 排名、Amazon's Choice 徽章 |
| 稀缺 | "仅剩 3 件"、Lightning Deal 倒计时、Prime Day 独占 |
| 权威 | Verified Purchase、品牌备案、Vine 计划、专利标注 |
| 互惠 | Coupon、首单立减、Subscribe & Save、赠品 |
| 承诺一致 | 心愿单、Follow 品牌、订阅制 |
| 喜好 | 品牌故事视频、A+ 页面情感化文案、KOL 合作 |
| 团结 | 母婴专属、DIY 认证、可持续认证 |
| 损失厌恶 | "错过今天恢复原价"、限时免邮 |
| 默认助推 | Prime 默认勾选、一键下单、自动续订 |

## 5. 争议与开放问题

- **Agent 时代失效**：社会证明/稀缺/权威依赖"情绪反应"，AI Agent 结构化解析后情绪按钮失灵
- **跨文化差异**：集体主义 vs 个人主义下原则权重不同
- **"灰度"与"黑暗模式"**：原则被滥用于诱导性设计（dark patterns）
- **Agent 时代新六原则**：Agent 间的说服可能基于"可验证证据强度"而非心理捷径
