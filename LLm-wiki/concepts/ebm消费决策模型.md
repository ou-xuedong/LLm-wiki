# 🧠 EBM消费决策模型

## 1. 定义与核心原理

**EBM模型（Engel-Blackwell-Miniard Model）** 由 James Engel、Roger Blackwell 和 Paul Miniard 于 1968 年提出，是消费心理学中最经典的消费者决策过程模型，将消费者从萌生需求到购后评价的完整心理行为拆解为五个阶段。

**五阶段链条**：
1. **问题识别**（Need Recognition）：内外部刺激引发需求缺口感知
2. **信息搜索**（Information Search）：主动/被动收集候选方案
3. **方案评估**（Alternative Evaluation）：按多重属性对比候选
4. **购买决策**（Purchase Decision）：在特定情境下做出最终选择
5. **购后行为**（Post-Purchase Behavior）：使用体验、满意度、二次传播

**核心假设**：消费者是"有限理性"的问题解决者，受外部环境、内部心理、情境因素三方共同塑形。

## 2. 关键技术路线与变体

- **Howard-Sheth 模型**：强调重复决策中的学习过程，区分拓展式/有限式/常规式问题解决
- **Nicosia 模型**：聚焦品牌信息流对决策的影响路径
- **行为经济学扩展**（Kahneman、Thaler）：用 System 1/2 与助推（Nudge）替代纯理性假设
- **Customer Journey 演进**：AIDA → AISAS → McKinsey 5A（Aware-Appeal-Ask-Act-Advocate）

## 3. 行业应用场景

- **电商投流漏斗设计**：TOFU/MOFU/BOFU 三层广告策略直接对应前三阶段
- **亚马逊 Listing 构建**：主图打"问题识别"、五点+QA 打"信息搜索"、评论+A+ 打"方案评估"
- **CRM 与复购运营**：围绕"购后行为"设计 NPS、邀评、邮件召回
- **AI Agent 购物系统**：Agent 代理后，"信息搜索"与"方案评估"被压缩为机器并行推理，决策链被重塑（参见 [[零点击消费]]）

## 4. 代表性论文/项目

- Engel, Blackwell, Miniard (2006). *Consumer Behavior* (第十版)
- Kahneman (2011). *Thinking, Fast and Slow* — System 1/2 对 EBM 的认知革新
- McKinsey (2009). *The Consumer Decision Journey*
- 电商化实践：Amazon Flywheel 理论、Linkfox 决策推演模型

## 5. 争议与开放问题

- **线性假设**：实际消费者在五阶段间来回跳跃、并非线性推进
- **AI Agent 的改写**：当"信息搜索/方案评估"由 Agent 代理，EBM 中段是否仍属消费心理学问题
- **零点击消费挑战**：代理式决策下"购买决策"阶段的情境助推被清零
- **社交化决策**：短视频/KOL/社群对决策路径的打断效应
