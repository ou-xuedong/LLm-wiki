# OpenClaw

## 1. 定位与核心产品

OpenClaw 是一个 AI agent 产品集，核心使命是「让 agents 可组合、可复用、可审计」。旗下产品包括：

- **Clawsweeper**：AI 维护机器人，自动关闭无效 issues/PRs
- **Summarize**：Chrome 扩展 + CLI 工具，网页/Reddit/PDF 摘要

- 2026-04-26：ClawSweeper 上线，24 小时关闭 ~4000 issues
- 2026-04-27：Summarize 0.14.0 发布，集成 GPT-5.5 `--fast` 模式
