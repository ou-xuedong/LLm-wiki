# Cohere

## 1. 核心定位与业务模式
Cohere 是**企业级 AI 基础模型公司**，核心差异化：让企业在自己的私有云、VPC 或本地环境部署大模型，确保数据不出企业网络。与 OpenAI 的 API 模式不同，Cohere 主张"模型即服务，但数据主权归企业"。

**核心产品**：
- **Command 系列模型**：企业级文本生成/对话模型
- **Embed 模型**：业界领先的文本嵌入模型
- **Rerank 模型**：企业搜索重排序
- **North 平台**：一站式企业 AI 工作流平台

## 2. 护城河 (Moat)
- **数据主权壁垒**：真正支持私有化部署，是金融、医疗、政府等数据敏感行业的首选
- **多云中立**：不绑定 AWS/Azure/GCP任意一家，避免厂商锁定
- **模型定制能力**：企业用自己的私有数据微调专属模型，这是监管行业的硬需求
- **Command R 系列**：专注企业 RAG 场景，检索增强生成能力突出
- **强大的搜索/嵌入能力**：Cohere Embed 是很多企业知识管理系统的底层依赖

## 3. 命门与软肋 (Weaknesses)
- **资金压力**：相比 OpenAI、Anthropic，Cohere 的融资金额和估值相对较低，竞争需要持续烧钱
- **模型能力差距**：基础模型能力与 GPT-4、Claude 的差距是否在缩小，有待观察
- **企业 AI 泡沫**：如果企业对 AI 的投入缩减，Cohere 的高客单价模式可能受冲击
- **开源竞争**：Llama 等开源模型的能力在追赶，企业是否还会为闭源模型付费

## 4. 重大事件与动态
- [2026] North 平台 GA，成为企业 AI 落地的一站式选择
- [2025] 持续深耕金融和医疗客户，医疗合规部署案例增加
- [2024] 发布了 Command R+ 和 Embed v3，企业采用率显著提升
