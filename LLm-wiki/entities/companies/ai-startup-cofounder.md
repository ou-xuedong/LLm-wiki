# ai-startup-cofounder

## 1. 核心定位与业务模式

AI-powered startup co-founder platform，配有 **13 个专业 AI Agent**，分工覆盖创业全流程。

核心理念：让创始人拥有一个"全能 AI 联合创始人团队"，不需要招募那么多人。

## 2. 核心架构

### 13 个专业 Agent
覆盖创业各个维度：
- 战略 / 市场 / 产品 / 技术 / 法务 / 财务 / HR / 运营 等专业分工

### 技术栈
- **前端**：Next.js / React
- **Agent 编排**：自研多 Agent 协调层
- **后端**：Node.js / Python（推测）

## 3. 与主人方向关联

- **13 Agent 分工模式**：可参考投流 Agent 的专业分工设计（每个投流决策维度一个 Agent）
- **全栈覆盖**：一人公司 + AI Agent 替代多角色联合创始人 → 投流 Agent 同样是"替代多个投流专家角色"
