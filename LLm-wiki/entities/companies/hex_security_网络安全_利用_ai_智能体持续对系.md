# Hex Security（AI 持续网络安全检测）

## 1. 核心定位与业务模式
Hex Security 是一家专注于**用 AI Agent 持续监控和检测企业网络/系统安全**的公司。与传统安全工具依赖规则和签名库不同，Hex 的 AI Agent 能够持续学习系统正常行为，自动发现异常模式，并实时触发响应。定位：企业安全运营的 AI Agent 化。

*注：官网信息有限，以下内容基于赛道通用逻辑推断，建议后续补充核实。*

## 2. 护城河 (Moat)
- **持续监控架构**：AI Agent 7x24 运行，相比人工轮询效率更高
- **异常检测智能化**：不依赖预定义规则，而是从数据中学习正常基线
- **快速响应闭环**：检测到威胁后自动触发响应动作，减少人工介入

## 3. 命门与软肋 (Weaknesses)
- **赛道拥挤**：安全可观测性和威胁检测领域已有大量玩家（Splunk、CrowdStrike、Darktrace）
- **AI 误报问题**：AI 异常检测的误报率直接影响运营团队效率，是所有这类产品的共同痛点
- **信息不足**：官网无法访问，实际产品能力难以评估，需进一步核实
