# employee-number-two（Claude Code AI 联合创始人）

## 1. 核心定位与业务模式

Your AI co-founder in Claude Code。核心理念：在 Claude Code 环境内，AI 作为"二号员工"全程参与创业。

**最大特色**：Sequential Projects——多项目串联工作流，从假设验证一路用到融资准备。

## 2. 核心工作流

```
假设验证 → 原型构建 → 市场测试 → 融资准备
```

每个阶段由专业 Agent 承接，输出直接作为下一阶段输入，形成完整创业链条。

## 3. 与主人方向关联

- **Sequential 工作流**：投流 Agent 也可以设计"诊断 → 策略 → 执行 → 复盘"的串行工作流
- **Claude Code 内运行**：与投流 Agent 的 CLI/TUI 方向一致，强调工具化而非聊天化
- **融资准备 → 可类比 → 投流复盘**：两者都是"周期性复盘 + 下一轮优化"
