# Wedge（AI 安全可观测性与 Agent 自动化平台）

## 1. 核心定位与业务模式
Wedge 定位为 **"用 AI Agent 重塑企业安全可观测性"**，核心能力：用 AI Agent 自动监控生产环境中的 AI 系统和大模型应用，实时检测异常、预测风险、自动修复。不同于传统安全工具靠规则和阈值，Wedge 的 AI Agent 能够理解上下文、判断正常/异常行为的边界，并自动执行响应动作。

**关键数据**：
- 覆盖 25+ 行业
- 100+ Agent 模板
- 声称实现 10x ROI
- 7x24 全天候自动化运行

## 2. 护城河 (Moat)
- **AI Agent 原生架构**：安全监控本身就是 Agent 在运行，不是传统规则的 AI 包装
- **多行业模板积累**：100+ 预设 Agent 模板覆盖 25+ 行业，工程实施效率高
- **极速部署**：声称可以在"几分钟内"上线，无需数月的实施周期
- **全天候自动化**：解决了企业安全团队人手不足的痛点

## 3. 命门与软肋 (Weaknesses)
- **品牌知名度有限**：相比 Lakera、Wedge 等知名安全品牌，Wedge 在市场上的声量较小
- **大厂竞争**：Splunk、Datadog 等可观测性巨头都在加 AI 功能，可能被平台化
- **安全 Agent 仍不成熟**：AI Agent 在安全领域的应用仍处于早期，技术成熟度有待验证
- **融资情况不明**：公司规模和资金实力不明，长期竞争力存疑

## 4. 重大事件与动态
- [2026] 持续扩展 Agent Economy 生态，推出更多行业模板
- [2025] 产品正式 GA，进入商业化阶段
