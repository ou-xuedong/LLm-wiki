# Exa

## 1. 核心定位与业务模式
Exa 是**面向 AI Agent 的神经网络搜索引擎**，提供 Web Search API 和 AI 原生搜索能力。与传统关键词搜索不同，Exa 使用深度学习模型理解查询意图，返回语义相关的文档片段（Highlights），专为 LLM 和 Agent 提供"事实 grounding"。核心价值：让 AI 不再胡说八道，用真实网络上最新鲜的事实回答问题。

**特色功能**：
- **神经网络索引**：不是 PageRank，而是真正理解内容语义的搜索
- **Highlights**：从页面中提取最相关的片段，直接喂给 LLM，减少 token 消耗
- **多垂类索引**：人物、公司、代码、金融、新闻等专项索引
- **零数据保留（ZDR）**：可选按需清除查询数据，满足隐私合规

## 2. 护城河 (Moat)
- **AI Native 定位**：专为 LLM 和 Agent 设计，不是改造传统搜索，而是从底层重做
- **模型无关**：支持 OpenAI、Anthropic、Gemini 等所有主流 LLM，不被某一模型绑定
- **高质量索引**：已获 OpenRouter、Anara 等 AI 公司认可为最佳 grounding 方案
- **Benchmark 领先**：在准确性、延迟等关键指标上持续领先
- **SOC 2 Type II 认证**：企业级安全合规，打开 B2B 市场

## 3. 命门与软肋 (Weaknesses)
- **搜索是红海**：Google、Perplexity、Tavily 都在做同一个市场，竞争极其激烈
- **数据新鲜度依赖爬虫**：高质量实时信息获取能力是否持续领先是关键
- **API 定价压力**：随着 LLM 厂商自建搜索能力，Exa 的独立价值可能被稀释
- **极低成本替代风险**：未来某搜索 API 可能以极低价格冲击市场

## 4. 重大事件与动态
- [2026] 持续在 Agent Search 赛道领跑，被 OpenRouter 等主流 AI 平台集成
- [2025] 发布多垂类索引（代码、金融、新闻），扩展到企业知识管理场景
- [2025] SOC 2 Type II 认证完成，企业客户数快速增长
