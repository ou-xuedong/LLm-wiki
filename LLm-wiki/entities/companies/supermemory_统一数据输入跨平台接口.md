# SuperMemory（个人第二大脑）

## 1. 核心定位与业务模式
SuperMemory 定位为 **"个人第二大脑"**，核心能力：聚合用户散落在各个平台（Twitter、Readwise、Kindle、Notion、Chrome 等）的所有知识碎片，用 AI 建立语义索引和知识图谱，让用户能够像"向人提问"一样检索自己的所有记忆。

**核心问题**：人们的知识散落在 50+ 个工具里，找东西的时候往往"记得见过但找不到"——SuperMemory 用 AI 终结这个痛苦。

## 2. 护城河 (Moat)
- **跨平台数据聚合**：已支持 50+ 数据源，是市场上集成最多的知识管理 AI 之一
- **知识图谱而非向量检索**：不只是语义相似度匹配，而是建立概念之间的关联网络
- **个人 AI 记忆**：与通用 AI 不同，SuperMemory 是"你的"AI，只学习你自己的数据
- **"第二大脑"品牌心智**：品牌定位清晰，用户一想到"要找到以前看过的东西"就想到 SuperMemory

## 3. 命门与软肋 (Weaknesses)
- **数据同步的脆弱性**：依赖第三方 API 同步数据，任何平台的 API 变更都会影响体验
- **大厂跟进风险**：Notion AI、Obsidian（Clipboard extension）、Evernote 都在做类似功能
- **使用频率存疑**：知识管理工具的"第二大脑梦想"很美好，但用户实际使用频率是否足够高？
- **隐私焦虑**：所有个人数据都聚合在一个地方，一旦泄露后果严重

## 4. 重大事件与动态
- [2026] 持续扩展数据源集成，推出团队版
- [2025] 推出 SuperMemory API，开发者可在自己应用里调用个人知识图谱
- [2024] Product Hunt 年度最受欢迎产品之一
