# Lark（飞书海外版）

## 1. 核心定位与业务模式
Lark 是字节跳动出海的 **企业协作与办公超级平台**，定位与飞书（中文版）相同，在海外市场与 Slack、Google Workspace、Microsoft 365 竞争。核心能力：即时通讯 + 视频会议 + 智能文档 + 项目管理 + AI 助手，一体化。比 Slack 更全能，比 Notion 更偏协作，比 Google Docs 更适合中文思维习惯（海外华人团队和亚洲公司首选）。

**注**：飞书/Lark 是字节跳动内部孵化的产品，知识库中有"飞书"标注，但实际运营主体是字节跳动，不是独立公司。此卡片记录字节跳动的 Lark 业务线。

## 2. 护城河 (Moat)
- **字节跳动生态加持**：抖音、TikTok、头条系产品在海外的数亿用户，为 Lark 提供天然的导入流量
- **极致效率整合**：将 Slack+Notion+Google Meet+Asana 多个工具合为一体，减少工具切换成本
- **AI 深度整合**：字节自研大模型（云雀等）赋能 Lark，AI 能力迭代迅速
- **价格竞争力**：相比 Microsoft 365 和 Google Workspace，Lark 定价更低，对 SMB 吸引力大

## 3. 命门与软肋 (Weaknesses)
- **地缘政治风险**：字节跳动背景，在欧美市场面临越来越多的监管审查（TikTok 问题的延伸）
- **品牌信任问题**：企业客户对字节跳动数据政策存疑，可能影响 Lark 进入大型企业
- **生态系统局限**：海外企业级 SaaS 生态深度不如 Slack/Microsoft，第三方集成有限
- **微软/Google 先发优势**：企业协作工具的切换成本极高，Lark 必须提供巨大差异化才能撬动成熟客户

## 4. 重大事件与动态
- [2026] Lark 在东南亚市场持续渗透，成为该地区增速最快的企业协作平台
- [2025] 推出 Lark AI，集成字节自研大模型能力
- [2024] 全球企业客户数突破新里程碑，海外华人企业使用率持续领先
