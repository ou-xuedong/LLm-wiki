# Claw Mart（AI 预制件交易市场）

## 1. 核心定位与业务模式
Claw Mart 是 **AI Agent 技能/预制件的交易市场**，核心问题：开发者开发的 AI 能力（如某个能完成特定任务的 Agent 模块）如何变现？Claw Mart 提供一个双方对接的平台——开发者发布并销售 AI 技能包（类似"AI 技能的 App Store"），使用者付费解锁并集成到自己的 Agent 工作流中。

**核心场景**：
- Agent 开发者将自己开发的垂直技能（如"简历筛选 Agent"）发布到市场
- 企业买家付费使用，无需从零开发
- 平台提供技能评级、交易担保、集成工具

## 2. 护城河 (Moat)
- **交易市场网络效应**：买家越多吸引越多卖家，卖家越多产品越丰富，形成飞轮
- **标准化封装**：如果 Claw Mart 能建立 Agent 技能的标准封装格式，就形成了平台壁垒
- **发现与评级机制**：类似 App Store 的评分体系，帮用户快速识别高质量技能

## 3. 命门与软肋 (Weaknesses)
- **技能标准化难题**：不同厂商的 Agent 接口不同，技能打包后的兼容性是大问题
- **冷启动挑战**：没有足够多的高质量技能，买家不会来；没有足够多的买家，卖家不愿来
- **大厂竞争**：如果 MCP 协议成为标准，OpenAI、Anthropic 可能直接在内层提供技能市场
- **监管风险**：AI 生成内容的版权和责任归属尚不清晰

## 4. 重大事件与动态
- [2026] EvoMap 与 Claw Mart 在知识库洞察 [[Agent技能分发与商业交易闭环]] 中被并列为 Agent 技能分发的两种代表模式
