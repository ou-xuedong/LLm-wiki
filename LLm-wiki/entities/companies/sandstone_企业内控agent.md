# Sandstone（企业法务合规 AI）

## 1. 核心定位与业务模式
Sandstone 是**面向企业法务部门的知识编排平台**，通过 AI Agent 连接公司内所有法律相关系统（合同库、邮件、Slack、CRM、HR 系统等），自动理解意图、收集上下文、路由任务。核心逻辑：让法务团队不再为"上下文税"买单——不再需要在割裂的系统里手动复制粘贴信息。

**典型场景**：合同审查 NDA、条款谈判、法律意见书自动生成、工作流路由。

## 2. 护城河 (Moat)
- **深度系统集成**：30+ 现成集成（Ironclad、Salesforce、Workday、Zip、Notion、Jira 等），开箱即用
- **法务场景专精**：不是通用 AI，而是专为法律工作流设计的知识编排引擎，有深厚的legal-tech 积累
- **AI 原生架构**：不是把传统软件加 AI 外壳，而是从底层重新设计 Agent 工作流
- **客户粘性极高**：法律部门对供应商切换极其谨慎，一旦工作流跑在 Sandstone 上，迁移成本极高

## 3. 命门与软肋 (Weaknesses)
- **市场窄众**：只做企业法务，市场规模相对有限；大厂可能推出同类功能（Notion AI、Copilot 等）
- **部署复杂度**：30+ 系统集成意味着实施成本高，Sales Cycle 长
- **数据安全要求极高**：法务数据极其敏感，大型企业可能要求私有化部署，增加销售阻力
- **竞争对手环伺**： Harvey（法律 AI）、Ironclad（合同管理）都在向 Agent 方向延伸

## 4. 重大事件与动态
- [2026] 持续深耕企业法务 AI 市场，产品线扩展到"AI Redline"（合同逐条批注）
- [2025] 与 Zip（采购管理）、Workday（HR）等主流企业软件完成深度集成
