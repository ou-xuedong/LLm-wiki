# PPC Entourage Amazon PPC 工具

## 1. 核心定位与业务模式

PPC Entourage 定位为**利润优化导向的 Amazon PPC 管理工具**，帮助卖家从 ACoS/ROAS 等虚荣指标转向真实利润贡献。适合有动手管理意愿但想要更好工具的卖家，填补手动管理与全托管外包之间的空白。

2016 年成立，总部美国。域名现重定向至 SPS Commerce，疑似已被收购，现属于 Carbon6 生态系统一部分。

**变现方式：** 早期 SKU 层级订阅（$47~$247/月），现转为约月广告 spend 的 2.9% 抽成模式。

## 2. 护城河 (Moat)

- **True ACoS 利润追踪**：有机 vs PPC 流量对比，追踪真实利润贡献
- **精细化自动化**：支持 Dayparting、负向关键词自动化、批量竞价调整
- **多市场覆盖**：支持美国、加拿大、墨西哥、英国、德国、法国、意大利、西班牙、印度、日本 10 个市场
- **双账号模式**：同时支持 1P（Vendor）和 3P（Seller）账号
- **高评分口碑**：4.8/5 星（174 条评价），被称为"最佳教育内容"
- **半自动化灵活性**：Autopilot 模式可开关，满足不同控制欲用户

## 3. 命门与软肋 (Weaknesses)

- **已被收购/收购传闻**：域名跳转 SPS Commerce，最新动态不明
- **平台局限性**：专注 Amazon，缺乏多平台扩展
- **品牌势能弱于 Perpetua**：在大型品牌/聚合商中的认知度较低
- **功能深度vs广度**：在关键词研究等单项能力上弱于 Helium 10、Jungle Scout

## 4. 核心能力

- True ACoS 和利润追踪（Organic vs PPC 流量对比）
- 批量 Campaign 优化 + 每日自动化竞价调整
- 负向关键词自动化
- Dayparting（分时段出价）
- 关键词发现 + 匹配类型测试
- 后端搜索词优化 + Listing A/B 测试
- 高级 Autopilot 模式
- 支持 Sponsored Products / Brands / Display / Product Targeting
- Command Center 100+ 自定义指标看板
