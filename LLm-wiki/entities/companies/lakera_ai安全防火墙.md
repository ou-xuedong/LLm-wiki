# Lakera

## 1. 核心定位与业务模式
Lakera 是 **AI 原生安全平台**，专注保护 GenAI 应用免受 Prompt 注入、数据泄露、Jailbreak 等新型攻击。核心卖点：在不拖慢开发速度的前提下，给 AI 应用装上"安全气囊"。通过实时监测和拦截恶意输入/输出，让企业安心赋予 AI 真正的执行权。

**核心指标**：
- 保护全球 1M+ 黑客社区的威胁情报
- 实时监测 100+ 语言
- 亚 50ms 延迟

## 2. 护城河 (Moat)
- **威胁情报飞轮**：1M+ 白帽黑客社区贡献攻击样本，持续训练检测模型，形成数据壁垒
- **多语言实时监测**：100+ 语言实时分析，覆盖全球化场景
- **亚 50ms 低延迟**：安全检查不影响用户体验，解决了企业最担心的"安全拖慢 AI"问题
- **极速上市**：无需改变代码即可横向部署，降低企业接入门槛
- **Dropbox 等标杆客户**：签下 Dropbox 等行业巨头，品牌背书强

## 3. 命门与软肋 (Weaknesses)
- **大厂自带安全能力**：OpenAI、Anthropic、Microsoft 都在内置安全能力，可能逐渐替代第三方安全工具
- **安全边界模糊**：AI 安全与传统 AppSec 的边界在哪里？Lakera 的定位是否足够清晰
- **检测 vs 防御**：检测到攻击后如何处置？响应机制是否足够成熟
- **合规要求的不确定性**：EU AI Act 等监管要求是否会倒逼企业只能用某一类安全工具

## 4. 重大事件与动态
- [2026] 推出 MCP 安全保护模块，适配 Agent 时代的新攻击面
- [2026] 签约 Dropbox 作为旗舰客户，在企业市场快速扩张
- [2025] 从威胁检测扩展到"安全编排"，推出中央策略控制台
