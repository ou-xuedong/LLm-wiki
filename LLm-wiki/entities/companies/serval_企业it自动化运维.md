# Serval（AI 原生 ITSM 平台）

## 1. 核心定位与业务模式
Serval 是 **AI 原生的 IT 服务管理（ITSM）平台**，用自然语言处理员工的 IT 请求——比如"我的邮箱打不开了"→ Serval 的 AI Agent 自动诊断、发起工单、解决问题，无需人工介入 IT 运维人员。 Serval 定位：用 AI Agent 替代传统 ITSM 的复杂流程和工单系统，让 IT 运维团队专注于更高价值的工作。

**核心场景**：
- 员工用自然语言提交 IT 请求
- AI Agent 自动分类、优先级排序、路由到正确的处理人
- 常见问题自动解决，减少人工介入
- 知识库自动学习，持续提升解决率

## 2. 护城河 (Moat)
- **AI 原生架构**：不是把 AI 插件到传统 ITSM 上，而是从底层重新设计 IT 请求处理流程
- **自然语言交互**：员工无需学习工单系统，直接"说话"就能解决 IT 问题，体验远超传统 ITSM
- **自动化解决率**：随着知识库积累，AI 解决的问题比例持续提升，运维团队价值倍增

## 3. 命门与软肋 (Weaknesses)
- **ITSM 市场成熟**：ServiceNow、Freshservice 等传统 ITSM 巨头在市场中根深蒂固，Serval 需要证明自己的差异化
- **企业安全要求高**：IT 运维涉及企业内部系统，安全认证要求极高，可能拖慢企业采购
- **技术验证周期长**：企业更换 ITSM 系统的成本和风险都很高，销售周期极长
- **与 ServiceNow 竞争**：ServiceNow 已经在做 AI 功能，Serval 的独立价值是否足够？

## 4. 重大事件与动态
- [2026] Serval 发布 AI Agents for IT，产品正式商业化
- [2025] 获得种子轮/A 轮融资，开始服务第一批企业客户
