# 🏢 优时科技（具身智能 / 自动驾驶）

## 1. 核心定位与业务模式
专注于L4低速自动驾驶的公司，后延展至人形机器人领域，打造数据飞轮。解决低速封闭场景的物流/运输自动化问题，逐步向通用人形机器人演进。

## 2. 护城河 (Moat)
- **数据飞轮**：从L4自动驾驶积累的场景数据和运动控制能力，可迁移至人形机器人
- **多轮融资验证**：8家投资机构联合注资，说明资本市场对其技术路径的认可
- **从自动驾驶到具身智能的跨界扩展**：结合视觉AI与机器人硬件

## 3. 命门与软肋 (Weaknesses)
- 从L4低速自动驾驶到人形机器人的技术跨度大，需要完全不同的硬件和运动控制能力
- 人形机器人赛道竞争激烈（宇树科技等已先行），后发劣势明显
- 数据飞轮需要足够多的机器人部署量才能形成壁垒，初期冷启动挑战大

## 4. 重大事件与动态
- [2026-05-01] 完成数亿元B2轮融资，前海方舟领投，前海母基金、鲲翎资本、厚天资本等8家机构参与（来源：36氪）
- [2020-10] 获得英诺天使领投、驰星创投跟投的千万人民币天使轮
- [2019-07] 英诺天使领投千万人民币天使轮
- [2018-06] PNP中国领投数百万人民币种子轮
