# Manus（已被 Meta 收购）

## 1. 核心定位与业务模式
Manus 是通用 AI Agent，主打**全自动化浏览器操作与复杂任务执行**——用户只需给出目标，Manus 自动操控浏览器完成研究、设计、报告等任务。2026年被 Meta 收购，成为 Meta AI 产品家族成员，面向全球企业推广。

**核心产品线**：
- **Manus Browser Operator**：核心能力，全自动网页操作
- **AI Design / AI Slides**：AI 生成 PPT 和设计稿
- **Wide Research**：深度研究自动化
- **Mail Manus**：邮件自动化
- **Slack 集成**：协作工作流

## 2. 护城河 (Moat)
- **Meta 生态加持**：背靠 Meta，拥有全球最大的社交/企业分发渠道
- **通用性壁垒**：与垂直 Agent 不同，Manus 的通用浏览器控制覆盖所有 Web 场景，护城河极宽
- **并购后的资源**：大量工程资源注入，产品迭代加速
- **企业级覆盖**：已有 Team Plan、API、SSO 等企业功能

## 3. 命门与软肋 (Weaknesses)
- **监管压力**：被 Meta 收购后，可能面临欧美的反垄断审查，产品策略可能被迫调整
- **商业化路径**：B2B 企业市场渗透率仍有待验证，与现有 SaaS 巨头的集成能力是关键
- **Browser Agent 竞争加剧**：OpenAI、Anthropic 也在自建 Browser Use 能力，差异化被稀释
- **数据隐私争议**：Meta 的数据使用历史可能影响企业客户信任度

## 4. 重大事件与动态
- [2026] 被 Meta 收购，官方公告"Manus is now part of Meta — bringing AI to businesses worldwide"
- [2026] 推出 Team Plan 与 SSO，企业版正式商业化
- [2026] 开放 API（open.manus.ai），支持第三方集成
