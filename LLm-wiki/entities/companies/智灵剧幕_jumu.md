# 🏢 智灵剧幕（Jumu.zeelin.cn · AI短剧平台）

## 1. 核心定位与业务模式

中国第一个类似RentaHuman模式的AI短剧滴滴平台，推出A2H（AI派单给人）和A2A（AI派单给AI）两种新生产模式。基于[[zeelinclaw_多智能体商业平台]]构建，目标实现AI短剧的全流程自动化生产。

## 2. 护城河 (Moat)

- 先发优势：国内首个AI短剧滴滴平台
- A2A生产模式（AI派单给AI）实现全自动化

## 3. 命门与软肋 (Weaknesses)

- AI短剧质量瓶颈：文学剧本大模型和AI框架是核心瓶颈
- 2026年4月数据显示AI短剧广告流水已达真人短剧三分之一，传统短剧正在变成需要保护的对象

## 4. 重大事件与动态

- [2026-04-30] 累计孵化300余部AI短剧，累计曝光超30亿
- [2026-04-30] 推出下一代AI视觉内容生成引擎MetaViz元相（metaviz.zeelin.cn）
- [2026-04-30] 团队自制AI短剧能力已达100部/月
