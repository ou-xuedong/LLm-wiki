# Ramp（企业费用管理与 AI 审查平台）

## 1. 核心定位与业务模式
Ramp 是 **企业级全链路支出管理平台**，核心能力：企业信用卡 + 费用报销 + 供应商付款 + 采购审批 + 会计自动化，一体化。Ramp 的 AI 在背后自动审查每笔支出是否合规，在违规发生前就拦截，而不是事后报销时才发现问题。

**核心数据**：
- 50,000+ 企业客户（包括 Notion、Shopify、Webflow、Eventbrite 等）
- 客户累计节省：27.5M+ 小时
- 月结账周期缩短：75%
- 采购流程效率提升：3x
- 实施周期：30天内完成

## 2. 护城河 (Moat)
- **事前拦截而非事后审查**：AI 在消费发生前就设好规则，不是报销时才发现违规
- **企业信用卡锁定**：企业一旦大量使用 Ramp 信用卡，支付流水就跑在 Ramp 上，迁移成本极高
- **极速实施**：30天完成全系统部署，对比传统 SAP/Oracle 实施（6个月+）优势明显
- **极强品牌背书**：Notion、Shopify 等顶级 SaaS 公司是客户，形成口碑效应
- **5% 返现 + 合作伙伴优惠**：企业信用卡里返现力度最大之一

## 3. 命门与软肋 (Weaknesses)
- **信用卡费率压力**：Visa/Mastercard 费率监管风险，可能压缩利润空间
- **与银行竞争**：美国大银行（JPMorgan、Bank of America）都在推自己的企业卡和费用管理
- **国际化局限**：主要在美国市场，全球化扩展面临本地合规挑战
- **AI 误判风险**：AI 拦截正常支出可能影响员工体验，引发内部反弹

## 4. 重大事件与动态
- [2026] 推出 Ramp Agent，专为 AI 智能体设计的企业费用管理 API
- [2025] 企业客户突破 50,000 家，ARR 进入 SaaS 独角兽行列
- [2024] 推出 AI 驱动的采购审批流程（Intake-to-Pay），实现 3x 效率提升
