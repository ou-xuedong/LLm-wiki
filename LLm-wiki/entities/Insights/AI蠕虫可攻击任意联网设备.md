# AI 蠕虫：多伦多大学研究揭示新型攻击面

## 1. 核心论点

多伦多大学研究团队成功演示了一种 AI 蠕虫，可针对任意联网设备发起攻击。这表明 **AI Agent 的自主执行能力正在创造全新的网络安全攻击面**——传统网络防御假设攻击者是人类或脚本，但 AI 蠕虫具备上下文理解、自适应决策和规模化扩散能力，防御范式需要根本性升级。

## 2. 关键信息

- **研究机构**: University of Toronto（多伦多大学）
- **发布日期**: 2026-06-03
- **核心发现**: AI 蠕虫可利用 Agent 的工具调用能力和联网特性，在设备间自主传播
- **影响范围**: 任意联网设备——从 IoT 到云服务器均可能成为目标

## 3. 趋势判断

AI Agent 安全将从"Prompt 注入防御"升级为"行为级威胁检测"——类似传统 EDR（端点检测与响应）但面向 Agent 的自主行为模式。这一方向可能催生新的安全品类：Agent Detection & Response（ADR），与当前 Lakera、Alter 等公司的 prompt 层防御形成互补。
