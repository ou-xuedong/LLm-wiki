## 2. 核心研究范式：计算建模 + 神经验证

陈琦研究的核心方法论是将**强化学习（RL）计算模型**与**神经影像（fMRI/EEG）**和**神经调控技术（HD-tDCS）**三者结合，构建"计算-神经-行为"闭环，核心范式如下：

```
概率反转学习任务 (Probabilistic Reversal Learning Task)
    ↓ 被试行为数据
RL 计算模型拟合 → 估计个体化参数
    ↓
参数与脑区激活的相关分析（fMRI/EEG）
    ↓
神经调控因果验证（HD-tDCS 增强/抑制目标脑区）
    ↓
计算级解释：认知功能的神经计算机制
```

**为什么用概率反转学习任务**：该任务模拟了现实中最核心的认知挑战——规则在不知会的情况下突然反转，被试必须快速检测变化、估计新规则、切换行为。这直接对应"认知灵活性"的核心定义。

## 3. 研究方向一：认知灵活性（Cognitive Flexibility）

### 3.1 学术定义

认知灵活性指个体在环境规则发生变化时，能够及时检测到变化、估计新规则并相应调整行为的能力。这是"适应性决策"的核心，是人类在不确定环境中生存的基础认知能力。

### 3.2 神经机制：陈琦的因果发现（2025, The Journal of Neuroscience）

陈琦团队 2025 年发表在神经科学顶刊 *The Journal of Neuroscience* 的论文，提供了认知灵活性的**首个因果证据**，揭示了完整的中介链条：

**核心发现**：

```
中额叶皮层（Medial Frontal Cortex, MFC）的 θ振荡（4-8Hz）
    ↓ 调节
规则预测误差（Rule Prediction Error, RPE）信号
    ↓ 驱动
规则切换（Rule Switching）行为
```

**研究设计**：
- 48 名健康成人（18 女，30 男）
- HD-tDCS（高密度经颅直流电刺激）：阳极刺激增强 MFC θ振荡 vs 假刺激对照
- EEG 记录 θ振荡功率
- 概率反转学习任务
- RL 计算模型估计规则预测误差

**关键结果**：
| 条件 | MFC θ功率 | 规则预测误差 | 规则切换所需试次 |
|---|---|---|---|
| 阳极刺激（增强θ） | ↑ | ↑ | 减少（切换更快）|
| 假刺激对照 | 基线 | 基线 | 基线 |

**结论**：阳极刺激降低了 MFC θ功率以及规则预测误差的幅度，同时增加了规则反转后稳定切换到新规则所需的试次数。证明：**θ振荡通过放大规则预测误差信号来驱动认知灵活性**。

### 3.3 计算模型层面的机制解释

陈琦的认知灵活性计算模型核心是一个**分层 RL 系统**：

```
刺激输入
    ↓
两套并行评估系统：
    ① 目标导向系统（Goal-directed）→ 基于模型（Model-based）→ 前额叶
    ② 习惯系统（Habitual）→ 无模型（Model-free）→ 纹状体
    ↓
两套系统的输出进行竞争/加权
    ↓
最终动作选择
```

- **认知灵活性高的人**：能够根据环境变化率动态调整两套系统的权重——稳定时习惯系统主导（省能），变化时快速切换到目标导向系统
- **认知灵活性低的人**（如 OCD 患者）：习惯系统权重异常偏高，无法根据环境变化调整

## 4. 研究方向二：计算精神病学（Computational Psychiatry）

### 4.1 方法论概述

计算精神病学是用**强化学习计算模型**来理解、诊断精神疾病的新范式。其核心思想：把精神症状重新定义为"认知算法参数的异常偏差"，从而实现量化诊断和精准干预。

**传统精神病学 vs 计算精神病学**：

| 维度 | 传统方式 | 计算精神病学方式 |
|---|---|---|
| 诊断依据 | 症状描述（主观） | 模型参数偏差（量化） |
| 机制解释 | 相关性描述 | 因果模型 |
| 个体差异 | 分类粗（正常/异常） | 连续参数（个性化） |
| 预测能力 | 弱 | 强（可验证）|

### 4.2 陈琦的核心研究成果

#### 成果 A：OCD 的计算模型机制

陈琦团队系统性地用 RL 模型解析了强迫症（Obsessive-Compulsive Disorder）的认知缺陷，核心论文包括：

**论文 1**（2022, *PLoS Computational Biology*）：**OCD 患者 Pavlovian 习惯对工具性行为的异常影响**
- 发现：OCD 患者的习惯系统（由 Pavlovian 信号驱动）对目标导向工具性行为的压制作用**减弱**
- 机制：正常人中，威胁相关刺激会激活习惯性回避；但 OCD 患者这种自动化保护机制失灵，导致强迫检查/洗涤行为无法被目标导向系统抑制

**论文 2**（2019, *Psychological Medicine*）：**OCD 患者目标导向学习系统的损伤生物标志物**
- 发现：OCD 患者在纹状体（尾状核/壳核）层面的功能与结构连接异常，导致目标导向系统无法有效覆盖习惯系统
- 方法：fMRI + RL 模型参数估计
- 意义：首次在活体上验证了 OCD "认知去灵活性"的神经环路基础

**论文 3**（2022, *NeuroImage: Clinical*）：**OCD 患者的尾状核与壳核连接失衡**
- 发现：OCD 患者的尾状核（与习惯系统强连接）和壳核（与目标导向系统强连接）之间的功能连接模式异常，且这种异常在患者的一级亲属中也存在，提示是内表型

#### 成果 B：抑郁与自我损伤的计算模型

**论文**（2022, *Brain and Behavior*）：**抑郁和非自杀性自伤患者的目標导向控制功能障碍**
- 发现：抑郁患者的目标导向系统活动减弱，表现为无法根据负面结果更新行为策略
- 计算机制：负向预测误差（negative PE）信号的神经表征减弱，导致学习率（α）异常

#### 成果 C：跨精神疾病的计算模型框架

陈琦将计算精神病学方法论系统化，提出了一个可应用于多种精神疾病的统一框架：

| 疾病 | 核心参数异常 | 神经表征 | 行为表现 |
|---|---|---|---|
| OCD | λ（习惯权重）↑ | 纹状体功能异常 | 明知无用仍重复 |
| 抑郁 | α（学习率）↓ 或异常 | 前额叶-MFC 激活减弱 | 无法从负面反馈中学习 |
| ADHD | γ（折扣因子）↓ | 前额叶功能不足 | 冲动、只看眼前 |
| 成瘾 | γ（折扣因子）↓↓ | 多巴胺系统异常 | 决策短视 |

### 4.3 计算模型的具体数学形式

陈琦使用的 RL 计算模型的核心是**Actor-Critic 架构**的神经实现：

**价值估计（Critic）**：
```
V(s) ← V(s) + α · δ
δ = r - V(s) - θ · θ（预测误差 + θ振荡调节项）
```

**策略更新（Actor）**：
```
π(a|s) ← π(a|s) + α · δ · (∂lnπ(a|s)/∂θ)
```

**习惯化模块（Pavlovian）**：
```
habit_strength = λ · Pavlovian_signal(s)
final_output = (1-λ) · goal_directed + λ · habit_strength
```

其中 λ 是陈琦在 OCD 研究中最关键的发现——习惯权重的异常偏高。

## 7. 对应主人创业方向的合作切入点

### 切入点一：探索-利用（Exploration-Exploitation）切换机制

**陈琦研究的认知本质**：人在"利用已知最优策略"和"探索新策略"之间的切换能力，取决于 MFC θ振荡调节的规则预测误差信号。

**对应投流场景**：运营每天面临的经典困境——"预算该继续放大当前爆款关键词（利用），还是测试新词新人群包（探索）？"

**合作价值**：陈琦的 RL 模型可以给 Agent 提供"动态切换阈值"——不是靠固定规则，而是根据当前环境的"规则稳定程度"自适应调整探索率。这比现有市面上的"随机分配20%预算测新词"的规则引擎先进一代。

### 切入点二：可解释的"认知诊断"能力

**陈琦的计算精神病学框架**：把认知状态变成一组可量化的 RL 参数（α、λ、γ等）。

**对应投流场景**：运营决策失误的原因，过去只能归为"经验不足"或"状态不好"。有了认知诊断框架后，可以具体指出："该运营的 λ（习惯权重）偏高，倾向于重复已过时策略；建议打开 x% 预算用于强制探索"。

**合作价值**：这种"认知诊断"能力，既是差异化的产品功能，又是可以发表论文的学术成果。

### 切入点三：预测误差（PE）触发的主动干预

**陈琦的研究核心发现**：认知灵活性的触发机制是"规则预测误差"——当环境给出的反馈超出预期时，系统自动激活目标导向系统。

**对应投流场景**：Agent 可以监测"预测准确度"——当某个策略的实际 ROAS 持续偏离预测值（PE 累计增大），自动触发"环境已变、重新评估"的信号，而非继续执行已失效的策略。

## 8. 方法论对 AI Agent 工程的启发

陈琦的工作对投流 Agent 的架构设计有直接启发：

| 认知科学概念 | AI Agent 对应 | 工程实现 |
|---|---|---|
| 目标导向系统（Model-based） | 策略评估模块（预测结果）| 模拟环境模型 |
| 习惯系统（Model-free） | 快速动作策略（已有经验）| Q-learning 价值函数 |
| 规则预测误差（RPE） | 策略-结果偏差检测器 | PE = actual_ROAS - expected_ROAS |
| MFC θ振荡（PE放大器） | "警觉"信号（当 PE 超过阈值）| 触发探索模式 |
| 认知灵活性 | 动态探索-利用平衡 | 自适应 ε-greedy |
| 习惯权重 λ 异常偏高 | Agent 陷入局部最优 | 检测并触发重置 |
