# 李剑锋（机智流）

## 基本信息

- 身份：AI 大模型技术讲师，自媒体"机智流"主理人
- 领域：大模型微调（SFT、继续预训练、LoRA、QLoRA）、LLM 训练工程
- 平台：微信公众号「机智流」

## 核心教学体系

围绕大模型微调与训练，已构建完整课程体系：

1. SFT 全量微调入门实战
2. 基于 Transformers 的参数高效微调（LoRA）
3. QLoRA：小显存环境大模型高效微调
4. 自我认知微调（知识蒸馏）
5. **从 SFT 到继续预训练**：大模型领域知识注入原理与实战 ← 本次摄入

## 主要观点

- SFT 更擅长行为模式调整，而非真正注入领域知识
- 继续预训练是垂直领域知识注入的正确路径，但成本高，非默认选项
- 垂类知识注入需要：高质量领域语料 + 合理数据配比实验 + 后续任务适配
- QLoRA 是降低继续预训练门槛的有效手段
