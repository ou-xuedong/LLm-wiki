# Cat Wu (Cat Wu)
- **职位**：Claude Code 产品负责人（Product Lead, Claude Code），Anthropic
- **核心贡献**：主导 Claude Code 从六个月的交付周期压缩到一天，52 天内推出 74 项功能
- **引用来源**：Lenny's Podcast（2026-04-23）

## 核心观点

**速度哲学**："移除每一个阻碍上线的壁垒。"

**PM 角色转变**：Cat Wu 直言："过去很多功能从想法到上线要六个月，现在一个月，有些甚至一周或一天。要做到这一点，PM的关注点必须从'对齐多季度路线图'转向'如何最快把东西送出门'。"

**Side Quest 机制**：Claude Code 最受欢迎的几个功能（桌面版集成、主动询问用户功能、内置任务管理）都是这样诞生的——没人写过 PRD，就是某天下午有人想试试，同事们觉得好用，就直接上线了。
