## 三、Wikilinks 路径修正

旧知识库使用 `[[中文名_英文名_Title]]` 格式，链接文本与路径混在一起。

新 wiki 采用路径分离格式：`[[子目录/目标页面|显示文本]]`。

**修正策略（后续 lint 阶段执行）：**

1. 扫描所有 `[[...]]` 格式的 wikilinks
2. 比对新 index.md 确认目标文件是否存在
3. 若存在，将 `[[xxx_yyy_zzz]]` 替换为 `[[新路径/新slug]]`
4. 若不存在，保留原样，列入 broken links 报告

## 四、Frontmatter 标题修复

**已知问题：** 部分文件的 frontmatter `title` 只截取了文件名第一段（如 `Andrej` 而非 `Andrej Karpathy`）。

**修复方法：**
```python
# 读取每个文件的 H1 行，从 '# ' 之后提取完整标题
# 替换 frontmatter 中的 title 值
import re, yaml
for f in new_wiki.glob("**/*.md"):
    content = f.read_text()
    m = re.search(r'^#\s+(.+)$', content, re.MULTILINE)
    if m and m.group(1):
        h1 = m.group(1).strip()
        # 替换 frontmatter 中的 title
        content = re.sub(r'^title: .+$', f'title: {h1}', content, count=1, flags=re.MULTILINE)
        f.write_text(content)
```

### 5.4 Wikilinks 健康检查
```python
# 找出孤立页（无 inbound links）
import os, re
from collections import defaultdict

wiki = "~/Desktop/AILab/LLm-wiki"
outbound = defaultdict(set)  # page -> set of pages it links to
inbound = defaultdict(set)   # page -> set of pages linking to it

for root, _, files in os.walk(wiki):
    for fn in files:
        if not fn.endswith(".md"): continue
        path = os.path.join(root, fn)
        slug = os.path.relpath(path, wiki).replace(".md","")
        for m in re.findall(r'\[\[([^\]|]+)(?:\|[^\]]+)?\]\]', open(path).read()):
            outbound[slug].add(m)
            inbound[m].add(slug)

orphans = [p for p in outbound if p not in inbound]
print(f"孤立页数: {len(orphans)}")
for o in orphans: print(f"  - {o}")
```
