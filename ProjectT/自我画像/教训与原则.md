# 教训与原则

> 从经历中提炼的教训、思考总结、行事原则。每条注明来源（第几次/什么事），按主题归类。
> 决策咨询时与《自画像》一同必读。生命周期规则（成熟度升降级、引用义务、Lint）见 CLAUDE.md「原则生命周期」。

## 格式约定

每条原则固定字段：

```
### P-NN【成熟度】短标题
- 原则：一句话。
- 来源：第NN次 / 事件简述。
- 适用：什么场景下调用。
- 验证记录：被引用/被印证的逐条记录（暂无则留"（暂无）"）。
```

成熟度：候选 → 已验证 → 已确认（升降级条件见 CLAUDE.md）。

## 事业与决策

### P-01【候选】口头承诺不算数
- 原则：口头承诺不构成留下的理由，只认落地的岗位/文件/资源。
- 来源：第01次 / 华为数据工程负责人许诺 teamleader + DataAgent 挽留，全部未兑现，还搭上烂尾项目与续约危机。
- 适用：任何以"未来会给你 X"为条件的去留/接活决策。
- 验证记录：（暂无）

### P-02【候选】跳坑前先尽调
- 原则：追技术浪潮跳新坑前，先尽调团队与位置，再动身。
- 来源：第01次 / 2023 追 OpenAI 热转盘古 PM，连换两个团队、扛 B 绩效。
- 适用：被新浪潮吸引想换方向/换团队/创业时。
- 验证记录：（暂无）

### P-03【候选】不选看天吃饭模式
- 原则：不选收入主要由外部行情/风口决定的商业模式——与房贷和厌恶风险的现实不兼容。
- 来源：第02次 / A股量化 4 年：牛市赚熊市歇、本钱少收入不稳，最终心灰意冷退出。
- 适用：评估创业方向、副业、投资标的时的第一道筛子。
- 验证记录：（暂无）

### P-04【候选】警惕口号决策
- 原则：被一句话/一个浪潮点燃时，强制冷却并尽调后再动身。
- 来源：第02次 / "all in 大模型"一句感召即转盘古，落地即被坑；与 OpenAI 火爆转岗、Agent 浪潮欲创业同构。
- 适用：任何由"热点事件/口号"触发的重大变动冲动。
- 验证记录：
  - 第05次（跨场域印证）：一句有道德拉力的漂亮话即可推主人"为证明而松口"——"口号易感"不止对工作浪潮成立，对人际场域的情绪性话术同样成立，是元特征。
  - 第06次（日常建议引用）：作为"日常六建议"第五条直接产出——凡被一句口号点燃想立刻行动时强制冷却一天。原则可落到日常护栏层。
  - 第07次（再印证·升级候选→已验证）：主人抛"医疗最有价值因为能增寿命，想选它"——带道德光环的口号变体，与"all in大模型"同构，且叠加第06次"给方向找高大上正名"盲点。本原则在"创业方向选择"层再次命中，确认口号易感是稳定元特征。

### P-05【候选】一鱼多吃只吃认知层
- 原则：思路、方法论、踩坑经验可跨公司与个人项目共用；代码、设计文档、prompt、数据绝不共用，个人项目须全程自有设备、自有时间、独立仓库。
- 来源：第04次 / "Agent 自进化系统"与卟卟鸡、投流助手 caseBank 思路同源，讨论后确立隔离边界。
- 适用：在职期间推进任何个人项目/创业，及日后 BP/尽调材料措辞。
- 验证记录：（暂无）

### P-06【候选】个人沉淀为第一筛子
- 原则：在司接活以"是否留下个人沉淀"为第一筛子，汇报成本是第二位的。
- 来源：第04次 / 主人自述"不怕汇报，怕没沉淀的投入"；躲 DataOS、占 Agent 预研坑即此筛子的应用。
- 适用：分工、接活、占坑等一切在司投入决策。
- 验证记录：（暂无）

## 关系与沟通

### P-07【候选·证据减弱】绩效靠屋里有人说话
- 原则：绩效与资源是"屋里有人替你说话"分出来的，纯产出不会自动兑换。
- 来源：第04次 / 松波靠一句"不走"的承诺保 A 拿年终奖后离职——考核可被表态撬动的实证。
- 适用：任何指望"产出自动被看见"的预期；在司最后阶段＝主动让唯一关键人（R博）定期看见最低可见产出。
- 验证记录：
  - ⚠️ 第04次（澄清轮）降级：主人侧证据减弱——其 B 为道通新人首考（近默认值），不构成被亏待的强证据。松波侧实证仍在。

### P-08【候选】骗术窗口用过即关
- 原则：别人用过的骗术窗口即关闭——同一招第二个人用，代价翻倍。
- 来源：第04次 / 松波"承诺不走保A"得手后，R博对低承诺表态的警觉必然升高；主人的"预研买时间"必须有真东西兜底，与松波区分。
- 适用：评估任何"别人这么干成功了，我也这么干"的冲动。
- 验证记录：（暂无）

## 自我管理

### P-09【候选】情绪压倒系统，须外部纪律兜底
- 原则："急躁证明自己"的心态曾让成熟模型照样亏钱。重大执行期须有外部纪律（规则、复核人、冷静期）兜住情绪。
- 来源：第02次 / 量化期亏损复盘，主人自认。
- 适用：高压力、急于自证的阶段（如创业初期、新环境头 6 个月）。
- 验证记录：
  - 第04次观察侧面印证：从"沙盘求最优"到"全盘接受躺平"的策略切换由情绪事件（连收三个坏消息）触发而非新事实——同构再现。
  - 第05次（跨场域印证）：高情感卷入的私人事务中同样出现"反复求确认+主动加码+理性让位"，证明"情绪压倒系统"是跨场域元特征，非高压职场孤例——本原则适用面扩至一切高情感卷入决策。
  - 第06次（日常建议引用）：作为"日常六建议"第二条产出——凡不可逆决定情绪上头时推迟到次日+找即时复盘人过一遍。原则从"重大执行期"下沉到日常拍板层。

### P-10【候选】无兴趣动作当事实输入排计划
- 原则：把"我不会做没兴趣的事"当作计划的事实输入，而非待克服的缺点。凡成功必须依赖"不感兴趣但必要"动作（商务、关系、跟进）的计划，要么交给伙伴/结构承担，要么默认它不会发生并重排计划。
- 来源：第04次 / 主人自认"对没兴趣的事理都不想理"，30分钟低成本会前动作亦拒绝执行；6个月诅咒、绩效B同根。
- 适用：一切行动方案设计（创业分工、在司策略、跟进类任务），以及小昭出方案时的可行性校验——含"无兴趣动作"的方案视同不可执行。
- 验证记录：
  - 第06次（日常建议引用）：作为"日常六建议"第一条产出——无兴趣的事不立志去做，直接外包或写成自动规则。原则从方案设计层下沉到日常生活层（报税、运营、关系维护等）。

### P-11【候选】性格不可改，结构可改行为
- 原则：环境选择＞自我改造。任何重大行为目标一律配结构（守门人、冷却期、写进合同的分工），不配意志和决心；凡只靠"这次我会注意"支撑的计划视同必败。
- 来源：第04次 / 高自察低自改的定性；量化期明知急躁仍亏钱、明知诅咒仍重演4次。
- 适用：辞职、签约、转向等一切不可逆决策的执行设计。
- 验证记录：
  - 第05次（正面实证·升级候选→已验证候选）：有即时外部分析脚手架在场时，主人能当场放下"硬刚"姿态、快速换位转向并奏效——首次出现"结构在场→行为实时被矫正"的正向案例，印证"结构可改行为"非空谈。含义：脚手架须"在场且即时"才生效，事后提醒无用。
  - 第06次（日常建议引用）：作为"日常六建议"第四条产出——改习惯别立flag，要造环境（搭子/守门人/即时复盘工具）。再次强调"在场且即时"是生效前提。

### P-12【候选】缺点不掩盖，定价并外包
- 原则：用股权/分成/现金购买自己永远不会做的动作（商务、关系维护、重复运营），让渡的比例就是性格的价格——付这个价比假装自己会变便宜得多。
- 来源：第04次 / "如何扬长避短掩盖缺点"之问；B方向外贸老板主导商务即范例。
- 适用：创业股权设计、合伙分工、一切需要补短板的场合。
- 验证记录：
  - 第06次（收尾懈怠轮引用）：作为"收尾苦工外包"的依据——收尾打磨对主人＝无兴趣动作，花钱/分工买掉比自我改造便宜。原则从创业分工层扩到个人作品收尾层。

### P-13【候选】定稿前先问"还有什么没说的"
- 原则：方案定稿前先问"还有什么我不知道的关键事实"。主人的决策素材习惯分批浮现，首轮方案常建立在不完整事实上。
- 来源：第04次 / 七段补充信息，每段都推翻前一轮方案的前提（伪项目→同源→已分仓→未认真做→已另有口径）。
- 适用：小昭每次给重大建议前的固定动作；主人重大决策前的自查清单。
- 验证记录：（暂无）

### P-14【候选】关系场域忌用"求真"碾压，高情感卷入须挂冷静期
- 原则：进入人际/情感/关系场域时，"事>人、求真客观"那把尺会主动制造对立——分歧时先维护关系再讲逻辑，别用通则硬套关系型特例；凡高情感卷入、且结果不可逆的决策，强制挂"冷静期+外部复核人"，不靠当下情绪拍板。
- 来源：第05次 / 跨场域行为观察 O1（求真尺泛化伤关系）+ O2（情绪总开关私域成立）+ O3（回避拍板担责），源为非工作场域人际互动，细节私存外档。
- 适用：一切需要经营关系、给情绪价值的场合（创业商务、合伙、客户、私人重大抉择）；与 P-10"无兴趣动作当事实输入"互补——P-10 管"不会做"，本条管"做了也容易做反"。
- 验证记录：
  - 第06次（日常建议引用）：作为"日常六建议"第三条产出——和亲近的人起分歧先护关系再讲道理，通则别硬套关系型特例。原则落到日常人际层。

### P-15【候选】收尾懈怠＝认知假闭合，价值在最后10%而多巴胺在60%停供
- 原则：主人的动力随"我懂了/demo 验证"这一认知闭合点撤火，但成果价值常在最后 10% 的打磨。须把"demo 通过"识别为假完成信号，并用外部交付节点（交给真实用户/挂出去给人看）接管已撤掉的内在张力——不靠"再坚持一下"的意志。
- 来源：第06次 / 主人自陈"懂了就飘、剩下体力活就懈怠"，女儿画画收尾松懈同构；机制＝信息缺口好奇心+多巴胺预测误差+认知闭合。
- 适用：一切个人项目/作品/创业 MVP 的收尾阶段；与 P-10（无兴趣动作外包）、P-11（结构改行为）联用。
- 验证记录：
  - 第06次（ADD 鉴别拷问）：主人追问"是否注意力缺陷"，以"兴趣区超强专注（4年系统/6个月精通）反证注意力能力无缺陷、问题在燃料结构"完成鉴别——本原则的机制判断经受住一次正面拷问，区分了 ADD（能力缺陷）与本特质（动机/燃料结构）。

### P-16【候选】亲子场域忌贴自身性格标签，夸过程不夸天赋
- 原则：不把自己的性格模式标签贴到孩子身上（"她跟我一样"）——儿童行为多为发育常态，过早认领会成自证预言。培养方向＝夸具体过程与打磨努力（Dweck 成长型思维）而非天赋/聪明，趁小种入"完成的愉悦"，正好对治"觉得自己会了就飘"的固定型思维。
- 来源：第06次 / 主人观察女儿画画收尾懈怠并急于认领同特征；与"拿性格当通行证"盲点同根。
- 适用：一切对子女/晚辈的特质判断与教养；亦提醒主人自身——洞察后要问"具体做什么"，避免标签消费。
- 验证记录：（暂无）

### P-17【候选】选方向用"个人适配尺"，不用"领域社会价值尺"
- 原则：评估创业方向时，唯一的尺是"真实需求 × 你能创造并捕获的价值 × 你扛得住的可行性"。领域对人类多崇高（增寿命、改变世界）与"你该不该选它"无关——"它很伟大"恰是最容易让主人松口的口号变体（与 P-04 合流）。先问"这领域里我能说出一个具体的人、具体的痛、愿掏的钱吗"，答不出即停在领域光环层，无真实需求。
- 来源：第07次 / 主人"想选医疗因为能增寿命"，被点破"领域社会价值 ≠ 个人适配"。
- 适用：一切创业/副业方向的领域级筛选；与 P-04（口号决策）、适配公式（赢在不可精通×产出归己×有人代办关系）联用；与"灯下黑"互查——主人已验证的强项（AI×科研提效）才是价值与擅长的交集。
- 验证记录：
  - 第07次（首次产出）：医疗踩主人"必输"区（强监管×准入关系×长周期），且逻辑矛盾（嫌金融封闭却奔更封闭的医疗）；据此判"医疗不作方向，AI×科研/医疗研究提效可作候选"。
  - 第08次（动机溯源印证）：主人自陈"喜欢思考意义（《长安的荔枝》）"——揭示"选医疗因伟大"背后是稳定的"意义驱动"特质（内部参照系）。意义驱动越强，越会持续把主人往高光环领域拉，故越需用个人适配尺把关。本原则的必要性被加固。

## 关系与价值反馈

### P-18【候选】分清"身份评价"与"市场反馈"——前者可不在乎，后者是生命线
- 原则：主人"不在乎别人评价"成立且是优点，但仅适用于**身份评价**（上司是否赏识、要不要站队、职级与面子）；**价值/市场反馈**（用户买不买单、产品好不好用、合伙人与客户的真话）是创业生命线，必须主动收、认真听、按它改。创业把两者绞在一起，**禁用一把"不在乎评价"通杀**——否则会把救命的市场信号一起屏蔽。
- 来源：第08次 / 主人自述"不太在乎别人评价、追寻内心" + "内部参照系"特质。
- 适用：MVP 市场验证、客户访谈、合伙沟通、demo 投放后的反馈采集；与适配公式"产出归己"配套——产出归己 ≠ 闭门造车。
- 验证记录：（暂无）

### P-19【候选】"这有什么意义"先验真伪——是真无价值，还是只是无趣了
- 原则：主人的"意义之问"有两副面孔——真价值筛（防为他人作嫁衣的无沉淀投入，正用，与 P-06 同源）／熄火的事后正名（事情进入收割期、重复期变无趣时，用"这有什么意义"把退场合理化，与"精通即厌倦"+"拿性格当通行证"同根）。触发此问时强制自查："是这件事本身没价值，还是我只是过了精通点不再有兴趣？"——后者不构成退场理由。
- 来源：第08次 / 《长安的荔枝》意义之问 + 既有"精通即厌倦／自察当麻醉"特质。
- 适用：一切项目/创业进入收割期、重复期的去留判断；与 P-15（收尾懈怠＝认知假闭合）、P-04（口号易感）联用。
- 验证记录：（暂无）

## 核心命题（待主人定向，定向后转为【已确认】原则）

- **"6个月诅咒"破除还是顺应**——精通即厌倦、获赏识即疏离的循环已重演至少 4 次（平安/华为流程IT/华为数据工程/道通），每次代价是边缘化或推倒重来。
